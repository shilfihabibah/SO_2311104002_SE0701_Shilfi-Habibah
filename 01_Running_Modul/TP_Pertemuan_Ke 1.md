## A. Dasar Teori

### a. Pengertian
Sistem operasi atau *Operating System* (OS) adalah perangkat lunak yang bertugas untuk mengelola sumber daya perangkat keras komputer sehingga bisa digunakan untuk menjalankan program aplikasi. OS akan muncul dan mulai beroperasi pada perangkat ketika proses booting atau ketika komputer dinyalakan. Komponen ini kemudian akan bekerja sebagai pengontrol utama interaksi antara hardware dalam perangkat dengan software yang ada di dalamnya. Secara keseluruhan, OS bekerja untuk memastikan performa semua jenis perangkat maksimal, baik dari segi kualitas maupun prosesnya, sehingga menyelesaikan kebutuhan pengguna dengan sempurna.
### b. Tools Pendukung
#### 1. Oracle VM VirtualBox
Suatu virtual machine kembangan Oracle. Mesin virtual ini memungkinkan kamu menggunakan berbagai jenis sistem operasi tanpa perlu ganti perangkat.
#### 2. Xinu OS
Sebuah sistem operasi ringan yang digunakan untuk tujuan praktikum ini.
#### 3. Ubuntu
Sistem operasi berbasis open source gratis yang multi fungsi dan relatif mudah digunakan. Ubuntu adalah hasil perkembangan dari OS Linux Debian, sehingga hampir semua library Linux dapat dijalankan dalam Ubuntu.
#### 4. Sourcetrail
Perangkat lunak source code explorer yang bersifat open-source, berfungsi untuk menampilkan struktur kode secara visual sehingga pengguna dapat dengan mudah memahami bagaimana setiap bagian program saling terhubung.
